# Gestor de Proyectos Colaborativos 

![MERN Stack](https://img.shields.io/badge/Stack-MERN-61DAFB?logo=react&logoColor=white)
![License](https://img.shields.io/badge/License-MIT-green)

Plataforma web para la **gestión colaborativa de proyectos** con integración de **Inteligencia Artificial**, diseñada para equipos que buscan optimizar su productividad mediante herramientas innovadoras y automatización inteligente.

---

## 🚀 Características Principales

### **Gestión de Proyectos**
- ✅ Creación, edición y eliminación de proyectos.
- ✅ Asignación de tareas con fechas límite, prioridad y seguimiento de progreso.
- ✅ Tableros Kanban interactivos para organización visual de tareas.

### **Comunicación en Tiempo Real**
- 💬 Chat integrado con soporte para mensajes, archivos y notificaciones push.
- 🔔 Alertas automáticas para actualizaciones críticas (ej: plazos próximos).

### **Panel de Administración**
- 📈 Dashboard con métricas visuales: progreso del proyecto, carga de trabajo por usuario y rendimiento del equipo.
- 🔐 Roles personalizados (admin, miembro, invitado) y gestión de permisos.
